MODUL UTAMA

Dua Pilar Kecerdasan

MPKB menggabungkan analisis CCTV real-time untuk kepatuhan dapur dan pemantauan kesehatan siswa secara berkelanjutan — didukung Claude AI di setiap lapisan.

🎥
Modul AI Dapur

Rekaman CCTV langsung dari zona dapur dianalisis terus-menerus oleh Claude AI untuk mendeteksi pelanggaran SOP, kegagalan kebersihan, dan risiko keselamatan — sebelum menjadi insiden.

  • Pemantauan CCTV langsung di Zona A, B, C
  • Claude Vision API mendeteksi: sarung tangan, jaring rambut, celemek, kebersihan
  • Daftar periksa SOP real-time — otomatis LULUS / GAGAL / PERINGATAN
  • Unggah video atau gambar untuk analisis AI instan
  • Log peringatan pelanggaran dengan stempel waktu
  • Skor kepatuhan harian dan laporan tren
  • Scan kualitas makanan melalui unggah gambar
  • Pelacakan & pengingat log suhu
🏫
Modul Kesehatan Sekolah

Pantau perkembangan fisik dan akademik setiap anak dari waktu ke waktu. Claude AI menghasilkan catatan kesehatan, menandai siswa berisiko, dan memberikan wawasan bagi guru dan orang tua.

  • Profil siswa: nama, usia, kelas, metrik kesehatan
  • Pemantauan tinggi, berat, BMI dengan indikator tren
  • Pemantauan nilai akademik per siswa
  • Catatan kesehatan buatan AI (kekurangan / kelebihan / normal)
  • Grafik tren BMI seluruh kelas per bulan
  • Dashboard penanda siswa berisiko
  • Rekomendasi program gizi melalui Claude API
  • Pembuatan laporan untuk orang tua & guru

✦ MENGAPA CLAUDE API?

Claude Sonnet 4 menawarkan keseimbangan terbaik antara kemampuan visi, kecepatan, dan biaya. Pada $3/juta token input, sistem pemantauan dapur yang menganalisis ~500 gambar/hari hanya menghabiskan sekitar USD $1,50/hari — sangat terjangkau untuk kepatuhan AI kelas enterprise.

Jalan Menuju Peluncuran

Rencana realistis 6 bulan dari kick-off hingga deployment penuh — dirancang untuk tim kecil dengan Claude AI sebagai intinya.

MINGGU 1–2
Setup Proyek & Arsitektur
Definisikan arsitektur sistem. Atur kunci Claude API. Konfigurasi skema database. Scaffold frontend React dan backend Node.js.
MINGGU 3–5
Integrasi CCTV Dapur
Hubungkan kamera CCTV. Implementasikan ekstraksi frame FFmpeg. Bangun panggilan visi Claude untuk pemeriksaan SOP. Sambungkan dashboard peringatan real-time.
MINGGU 6–8

Semua zona dapur dipantau. Penilaian kepatuhan aktif. Fitur unggah-untuk-analisis selesai. Pembuatan laporan harian berjalan.
MINGGU 9–12
Modul Kesehatan Sekolah

MINGGU 13–16
Pengujian & Pelatihan Staf
Pengujian end-to-end di kedua modul. Orientasi staf dapur dan admin sekolah. Pengumpulan umpan balik dan perbaikan bug.
MINGGU 17–24
Deployment Penuh & Go-Live
Deployment produksi ke AWS. Domain dikonfigurasi. Pemantauan sistem aktif. Sistem MPKB penuh beroperasi 24/7.

CLAUDE AI — LAPISAN KECERDASAN SISTEM
MPKB menggunakan Claude Sonnet API sebagai lapisan kecerdasan intinya. Untuk dapur, Claude menerima frame kamera berkode base64 setiap 30 detik dan diinstruksikan untuk mendeteksi pelanggaran SOP — sarung tangan hilang, penanganan makanan yang tidak tepat, kegagalan kebersihan. Ia mengembalikan JSON terstruktur dengan putusan, kepercayaan, dan penjelasan untuk setiap zona.

Untuk modul sekolah, Claude memproses metrik setiap siswa dan menghasilkan catatan kesehatan yang dipersonalisasi — mengidentifikasi apakah seorang anak kekurangan berat, kelebihan berat, atau sehat, dan merekomendasikan intervensi.

Setiap keputusan AI dicatat, diberi stempel waktu, dan dapat dijelaskan — memberikan administrator MPKB kemampuan audit penuh atas semua penilaian otomatis.
